El uso de las etiquetas de estructura.

Los estandares actuales nos proporcionan un conjunto de elementos HTML para englobar los contenidos
que se suelen encontrar en una pagina web.Cada uno de estos elementos esta pensado para contener
diferentes elementos HTML, como parrafos, imagenes, etc.

Funcion primordial

Interpretar de que forma tiene que presentar el texto
sepa de donde tiene que obtener las imagenes y sonidos y como mostrarlos
distribuya todos los contenidos de una forma determinada dentro de la pantalla
